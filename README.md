# django-library

django实现支持书籍增删改查.

## getting-started

运行app
```
docker run -d --name django-library \
 --restart always \
 -p 8000:8000 \
 willdockerhub/django-library
 ```
 
 浏览器访问app
 ```
 http://127.0.0.1:8000/books/
 ```
